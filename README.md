# Django Projects Repository

This repository contains two main Django projects:

1. **AntiGravity (`antigravity/`)**
   A full-stack web application built with Python + Django.
   See `agentsdjango.md` for detailed information about the architecture and guidelines for this project.

2. **Car Rental System (`car_rental_system/`)**
   A Django-based car rental application backend and frontend.

## Installation

```bash
pip install -r requirements.txt
```

## Running the Car Rental System

```bash
cd car_rental_system
python manage.py runserver
```

## Running the AntiGravity Project

```bash
cd antigravity
python manage.py runserver
```
